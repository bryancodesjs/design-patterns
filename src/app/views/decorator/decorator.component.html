Decorator Design Pattern

The Decorator Design Pattern allows you to add new functionality to an existing class, while preserving its original properties and methods.

Just think about it as a cake. In a cake, you have what's called the Base Cake, which is usually a light-brownish mass consisting of eggs, sugar, flour and baking powder.

Base cake

We can refer to this Base Cake as our base Cake class. Although it might taste good, our small bakery would soon go bankrupt if all we offer is spongy base cakes; We need to decorate the base with layers, colors, flavors and probably even add a custom message on top of the cake.

HBD Jenna!

The Decorator Design Pattern allows us to add this toppings and decorations without altering the Base Cake (after all, the client still wants a cake).
