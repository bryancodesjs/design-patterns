Context

In this example, we're simulating we are in a video game. On this view, we can add new Humans(characters) to our team by choosing from for types of humans:

  • Warrior
  • Scientist
  • Builder
  • Witch

There's a base interface called Human that defines the shape of a human object. In general, a Human object performs a default action, but we don't define the logic of the action on the human interface itself. The base Human interface only tells us a human is supposed to perform an action, but the actions of your chosen humans will vary based on the type of human you choose.

While a Warrior's action consists on slashing with a sword, a builder's action consists on building things, and so on.
